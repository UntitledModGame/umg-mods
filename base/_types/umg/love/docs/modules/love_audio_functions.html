newQueueableSource

  • newQueueableSource(samplerate: number, bitdepth: 8 | 16, channels: 1 | 2, buffercount?: number): Source
    • Defined in love.audio/functions.d.ts:145

    Creates a new Source usable for real-time generated sound playback with Source.queue.

    link

    love.audio.newQueueableSource

    since

    11.0

    Parameters

    • samplerate: number

      Number of samples per second when playing.

    • bitdepth: 8 | 16

      Bits per sample (8 or 16).

    • channels: 1 | 2

      1 for mono or 2 for stereo.

    • Optional buffercount: number

      The number of buffers that can be queued up at any given time with Source:queue. Cannot be greater than 64. A sensible default (~8) is chosen if no value is specified. (Default 0)

    Returns Source

    The new Source usable with Source.queue.
