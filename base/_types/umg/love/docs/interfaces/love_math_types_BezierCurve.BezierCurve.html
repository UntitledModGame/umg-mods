  • evaluate(t: number): LuaMultiReturn<[x: number, y: number]>
    • Defined in love.math/types/BezierCurve.d.ts:22

    Evaluate Bezier curve at parameter t. The parameter must be between 0 and 1 (inclusive).

    This function can be used to move objects along paths or tween parameters. However it should not be used to render the curve, see BezierCurve:render for that purpose.
