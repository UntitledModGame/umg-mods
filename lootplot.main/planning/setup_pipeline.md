


## ROUND -1  (perk-select)
player select perk.

Once selected, 3 items are spawned on the plot:
- perk-item
- worldgen-item
- doomclock-egg (ie gamemode item)


## ROUND 0  (WorldGen)
doomclock-egg spawns doomclock, next-round button, next-level button
perk-item spawns slots
worldgen-item spawns slots

## ROUND 1
Game begins!
